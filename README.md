# Bank Management System (Java)

## Description

A simple Java console application that allows users to create accounts, deposit money, withdraw funds, and check balances.

## Features

* Open account
* Deposit money
* Withdraw money
* Check balance

## Technologies

* Java
* OOP (Classes & Objects)
* ArrayList

## How to Run

```bash
javac Main.java
java Main
```

## Project Structure

* Main.java – runs the program
* Manager.java – handles operations
* Accounts.java – stores data
